# 🌱 Weed Detection AI

An AI-powered web application that detects weeds in crop images using deep learning (YOLO).

---

## 🚀 Live Demo

👉 https://weed-detection-2.onrender.com

---

## 🔗 GitHub Repository

👉 https://github.com/priyanshu4ai-hub/weed_detection

---

## 📸 Features

* Upload crop images via web interface
* Detect weeds using trained AI model
* Display bounding boxes on detected weeds
* Show confidence scores
* Clean and simple frontend UI

---

## 🖥️ Frontend

* HTML (Jinja Templates)
* CSS (custom styling)
* JavaScript (for API calls & UI interaction)

### UI Features:

* Image upload button
* Real-time result display
* Detection summary panel

---

## 🛠️ Backend

* Flask (Python web framework)
* Ultralytics YOLO (AI model)
* OpenCV (image processing)

## ⚙️ Installation (Local Setup)

```bash
git clone https://github.com/priyanshu4ai-hub/weed_detection.git
cd weed_detection
pip install -r requirements.txt
python app.py
```
